INSTALLATION INSTRUCTIONS:
--------------------------
1. Create database: CREATE DATABASE drupal7_db;
2. Import database: mysql -u root -p drupal7_db < sims_database.sql
3. Update sites/default/settings.php with database credentials
4. Set permissions: chmod 644 sites/default/settings.php
5. Clear cache: drush cc all
6. Access application: http://localhost/drupal7
